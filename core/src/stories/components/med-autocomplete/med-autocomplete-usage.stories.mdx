# Auto Complete

O componente Auto Complete deve ser usado como complemento do componente **ion-input**. O controle do componente deve ser feito pela tela onde esta sendo aplicado.

## Propriedades

| Propriedade  | Atributo | Descrição                            | Tipo       | Default  |
| ------------ | -------- | ------------------------------------ | ---------- | -------- |
