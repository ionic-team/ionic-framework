# Plus Minus

## Propriedades

| Propriedade  | Atributo    | Descrição                                   | Tipo      | Default       |
| ------------ | ----------- | ------------------------------------------- | --------- | ------------- |
| **dsSize**   | **ds-size** | Define a variação de tamanho do componente. | **"xl"**  | **undefined** |

## CSS Custom Properties

| Name               | Description                                   |
| ------------------ | --------------------------------------------- |
| **--color**        | Define a cor de texto do componente.          |

```
<monta-provas-plusminus>500</monta-provas-plusminus>
```
