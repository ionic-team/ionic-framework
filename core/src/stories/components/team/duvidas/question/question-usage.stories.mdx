# Question

## Propriedades

| Propriedade   | Atributo      | Descrição                                   | Tipo                                                                                                                 | Default       |
| ------------- | ------------- | ------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ------------- |
| **color**     | **color**     | Define a cor do componente.                 | **[Color](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined**   | **undefined** |
| **neutral**   | **neutral**   | Define a cor neutra do componente.          | **[Neutral](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined** | **undefined** |
| **collapsed** | **collapsed** | Define o estado do componente.              | **boolean**                                                                                                          | **false**     |
| **texto**     | **texto**     | Define o conteúdo de texto do componente.   | **string** \| **undefined**                                                                                          | **undefined** |

## CSS Custom Properties

| Name             | Description                                          |

| **--background** | Define a cor de background do componente.            |
| **--color**      | Define a cor do componente.                          |
| **--lines**      | Define o número de linhas visíveis com reticiências. |
