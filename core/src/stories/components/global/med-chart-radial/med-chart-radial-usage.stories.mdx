# Chart Radial

## Propriedades

| Propriedade | Atributo    | Descrição                        | Tipo                              | Default       |
| ----------- | ----------- | -------------------------------- | --------------------------------- | ------------- |
| **dsName**  | **ds-name** | Define a variação do componente. | **"secondary"**  \| **undefined** | **undefined** |
| **valores** | **valores** | Define os valores do componente. | **MedChartRadiaItem[]**           | **undefined** |

## Interfaces

```
export interface MedChartRadiaItem {
  cor: string;
  label: string;
  quantia: number;
  ignoreBarra: boolean;
};
```

## Default

```
<med-chart-radial></med-chart-radial>
```

## Variação Secondary

```
<med-chart-radial ds-name="secondary"></med-chart-radial>
```
