# Button

Documentação auxiliar ion-button: **https://ionicframework.com/docs/api/button**

## Default

### Propriedades

| Propriedade  | Atributo     | Descrição                                   | Tipo                                                                                                              | Default       |
| ------------ | ------------ | ------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- | ------------- |
| **dsColor**  | **ds-color** | Define a cor do componente.                 | **[MedColor](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined** | **undefined** |
| **dsName**   | **ds-name**  | Define a variação do componente.            | **"secondary"** \| **"tertiary"** \| **undefined**                                                                | **undefined** |
| **dsSize**   | **ds-size**  | Define a variação de tamanho do componente. | **"xxxs"** \| **"xxs"** \| **"xs"** \| **"sm"** \| **undefined**                                                  | **undefined** |
| **expand**   | **expand**   | Define a variação de tamanho do componente. | **"full"** \| **"block"** \| **undefined**                                                                        | **undefined** |
