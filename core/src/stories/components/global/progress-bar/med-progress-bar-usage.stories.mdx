# Progress Bar

## Propriedades

| Propriedade    | Atributo       | Descrição                                   | Tipo                                                                                                                   | Default       |
| -------------- | -------------- | ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- | ------------- |
| **dsColor**    | **ds-color**   | Define a cor do componente.                 | **[MedColor](http://desenv.ordomederi.com/templarios/?path=/story/enums-enums--page#medcolors)** \| **undefined**  | **undefined** |
| **dsName**     | **ds-name**    | Define a variação do componente.            | **"secondary"**                                                                                                        | **undefined** |
| **value**      | **value**      | Define o valor para progresso da barra.     | **number: > 0 <= 0.1**                                                                                                 | **undefined** |
| **percentage** | **percentage** | Esconde ou mostra a porcentagem.            | **boolean**                                                                                                            | **undefined** |

## Default

```
<ion-progress-bar value="0.3"></ion-progress-bar>
```

### Percentage

```
<ion-progress-bar value="0.3" percentage></ion-progress-bar>
```

## Variação Minimalist

```
<ion-progress-bar ds-name="minimalist" value="0.3"></ion-progress-bar>
```
