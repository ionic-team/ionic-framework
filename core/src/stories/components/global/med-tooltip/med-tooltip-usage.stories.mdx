# Tooltip

## Propriedades

| Propriedade   | Atributo      | Descrição                          | Tipo                                                    | Default       |
| ------------- | ------------- | ---------------------------------- | ------------------------------------------------------- | ------------- |
| **collapsed** | **collapsed** | Define o estado do componente.     | **boolean**                                             | **true**      |
| **content**   | **content**   | Define o conteudo do componente.   | **string \| undefined**                                 | **undefined** |
| **dsName**    | **ds-name**   | Define a variação do componente.   | **"definition" \| undefined**                           | **undefined** |
| **placement** | **placement** | Define a localização do componente.| **"bottom" \| "left" \| "right" \| "top" \| undefined** | **undefined** |
| **position**  | **position**  | Define a posição do componente.    | **"center" \| "end" \| "start" \| undefined**           | **undefined** |
| **titulo**    | **titulo**    | Define o titulo do componente.     | **string \| undefined**                                 | **undefined** |

## CSS Custom Properties

| Name             | Description                          |
| ---------------- | ------------------------------------ |
| **--background** | Define a cor da borda do componente. |
| **--z-index**    | Define o z-index do componente.      |
