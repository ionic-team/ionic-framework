# Image Zoom

O Image Zoom utiliza os componente modal e slides como base.

- Documentação auxiliar modal: **https://ionicframework.com/docs/api/modal**
- Documentação auxiliar slides: **https://ionicframework.com/docs/api/slides**

**As propriedades marcadas com ⚠️ são extritamente necessárias.**

```ts
const createModal = async () => {
  modalController.create({
    component: 'med-image-zoom', ⚠️
    cssClass: 'med-image-zoom', ⚠️
    componentProps:{ ... } ⚠️
  }).then((modal)=>{
    modal.present()
  });
}
```

## componentProps

```ts
componentProps: {
  titulo: string;
  imagens: [
    {
      src: string;
      legenda: string;}
  ];
  marcaAguaSuperior: string;
  marcaAguaInferior: string;
},
```

## Imagen Única

const createModal = async () => {
